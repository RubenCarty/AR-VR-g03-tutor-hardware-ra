# Tutor de Hardware de PC en Realidad Aumentada

## Descripción
Aplicación desarrollada en Unity con Vuforia que permite reconocer Hardware mediante realidad aumentada y mostrar información educativa sobre sus componentes.

## Objetivo
Facilitar la identificación de componentes de hardware como CPU, RAM y puertos mediante una experiencia interactiva en RA.

## Funcionalidades
- Reconocimiento de múltiple Hardware
- Visualización de etiquetas
- Botón para mostrar información
- Botón para ocultar información
- Descripción educativa del componente

## 🧪 Tecnologías utilizadas
- Unity
- Vuforia Engine
- C#
- Android

- Interacción básica implementada
